MongoDB

  1. Зарегался на оф сайте монго
  2. Создал бесплатный кластер
  3. При нажатиии CONNECT нам выдастся URL нашей базы и будет предожено создать пользователя и скачать MongoDB Compass - UI для управления базой данный.
  4. Скачиваем и устанавливаем MongoDB Compass
  5. Открываем MongoDB Compass, и коннектикмся к бд по нашему URL вставив в него login и password
  6. Cоздаем коллекцию, и кладем в нее наши объекты в виде json

Rest service

  1. Создаём Spring Boot приложение с двумя стартер пакетами: web и data-mongodb
  2. Добавляем в ресурсы URL нашей базы
  3. Создаём модель соответвующую модели в бд, и указываем к какому документу в БД относится наша модель
  4. Создаём слой досутпа к данным, унаследовавшись от MongoRepository и указав над которой моделью мы будем работать и тип id-шника
  5. Создаём контроллер доступа к данным
  6. Проверяем всё через Postman
  7. Можно еще тесты пописать

Deploy to Heroku

  • Регаюсь на Heroku
  • Выбираю Create new app
  • Придумываю уникальное название
  • После создания приложения, перехожу в него на вкладу Deploy, логинюсь на свой GitHub, выбараю приложение и ветку и нажимаю deploy. Приложение сразу начинает деплоиться
  • Переходим на вкладку Settings, и в разделе Domains видим URL на котором висит наш сервис
  • Проверяем через Postman, что все работает
